# Shule AI Plus +

Shule AI Plus + is a gamified learning platform designed for the Kenyan CBC (Competency-Based Curriculum). It provides a rich, interactive, and engaging educational experience for students, teachers, and parents.

## Description

This project is a web-based application that offers a wide range of educational resources, including games, worksheets, and lesson plans, all aligned with the CBC curriculum. The platform is designed to make learning fun and effective for students from pre-primary to junior secondary levels. It also provides valuable tools and resources for teachers to plan and deliver their lessons, and for parents to track their children's progress and support their learning at home.

## Features

- **Gamified Learning:** A wide variety of educational games covering all CBC subjects.
- **Comprehensive Resources:** A rich library of worksheets, lesson plans, and other educational materials.
- **Curriculum-Aligned:** All content is aligned with the Kenyan CBC curriculum.
- **Personalized Learning:** The platform provides resources for different grade levels, from pre-primary to junior secondary.
- **Teacher Support:** A dedicated "Teachers Hub" with lesson plans, assessment tools, and professional development resources.
- **Parental Involvement:** A "Parents Hub" to help parents track their child's progress and support their learning.
- **Tutor Marketplace:** A system for tutors to register and for students to find and request tutors.
- **Subscription-Based:** Offers flexible subscription plans for accessing the platform's premium features.
- **User-Friendly Interface:** A clean, modern, and easy-to-navigate user interface.

## Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You will need a modern web browser to run this application.

### Installation

1. Clone the repo
   ```sh
   https://github.com/memeyaiofficial-bit/ShuleAIv2.git
   ```
2. Open the `index.html` file in your web browser.

## Usage

The application is designed to be intuitive and easy to use. Users can navigate through the different sections of the application using the main menu. The main sections are:

- **Home:** The main landing page with an overview of the platform and its features.
- **Subjects:** A section with resources organized by subject.
- **Grade Levels:** A section with resources organized by grade level.
- **Games:** A collection of educational games.
- **Worksheets:** A library of printable worksheets.
- **Teachers Hub:** A dedicated section for teachers with lesson plans, assessment tools, and other resources.
- **Parents Hub:** A dedicated section for parents to track their child's progress and find resources to support their learning.

Users can also register as tutors or request a tutor through the platform.

## For Developers

This project is built using HTML, CSS, and JavaScript. The backend functionalities are handled by a Node.js server.

### File Structure

The project has the following file structure:
- `index.html`: The main entry point of the application.
- `css/`: Contains the CSS files for styling the application.
- `js/`: Contains the JavaScript files for the application's logic.
- `games/`: Contains the HTML files for the educational games.
- `Backend/`: Contains the server-side code for the application.
